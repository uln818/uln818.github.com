開站第一篇YO

沒啥原因，突然又想寫Blog了(笑)。

就像上個月初突然懷念起研究所時玩的網頁war game，懷念和各國阿斗啊的圓桌會議，順便想聊個天練英文，所以下載了Burtal Age狠狠了玩了一番 XD。

恩…是段相當有趣的經歷啊。

僅一個月，我們建立起了服器戰力最強的部族，WeChat群組中包含了德國人x2、美國人x2，西班牙人x2、中國人x2、台灣人x1(就是我)的領導層，整個部族將近100人，每天有討論不完的活動進攻策略。雖然之後因為我們的德國人盟主很不領導，使我們雖為第一，但卻一直輸給第二的盟(深刻體會領導力的重要啊)，不過整段過程還是相當的有趣~ 目前我已漸漸淡出，有機會再跟各位說這段奇幻體驗。

痾，回歸主題，為啥又想寫Blog？

Hm…想用Hexo + GitHub Page架站是去年的事了，主要是發現這架構完全免錢又超級方便，就隨手架了一下，恩，果然很爽，一整個順，優質體驗滿點。相關的套件方法法我大概年初就全部玩了一遍，只是沒那個Fu開始寫，直到最近感覺比較有空 + 買了m4yo網域不知道要幹啥 + 突然有Fu了。

我一直很喜歡Blog的社交節奏，討厭現今類FB等社交軟體。怎麼說? 距離感不同，你看你的我寫我的，不會即時跳出相互影響，只有文章、留言和回文，一來一往，皆在非同步中進行，互不干擾，舒服。再加上，現在的社交軟體怎麼了？訊息越弄越短，有的甚至只有貼圖，實在太速食主義拉，怎麼著~ 俺就是要寫長文！(ps.不過我還是很喜觀看大家的動態，所以請不要刪我FB好友~~)

第一次寫Blog，是距今快10年前的事了，該blog除了在我追老婆扮演重要角色外，也讓我和朋友們玩得十分開心，一共寫了四年。由於本人堅持不寫作做的專業文，只記錄開心歡樂的事，使得其中的文章，就算現在回去看，也是滿滿的回憶，Good啊，有興趣的可以看一下。不得不說本人當初選用blogger沒選錯啊，10年前的blog還可以活著到現在的大概只有blogger了。

這次重開寫作，主要有三件事我很想紀錄，這三件事每次發生就累積一點點，積到現在滿到不行，就…順著感覺走開寫唄。

第一當然是我兒 嚕嚕，雖說現在手機方便，容量大到不行，可是當你看著沒有重點只有日期的27G影片相片時，就會深深的體悟，沒整理過的回憶只是垃圾啊….。第二是食譜和健身資訊，哈哈，自從親自下廚紮紮實實持續煮了四個月的菜後 (三菜零湯，週一到週五下班回家煮，一週至少煮三天晚上)，發覺我還挺喜歡煮菜的，煮完了食譜該拍照記一記拉。至於健身嘛…我是館長鐵粉，他2000粉絲時我就加他了，你懂der。最後，是一些自己想玩 or 和m4小組員們玩的side project，嘴砲久了還是要自己動手做些好玩的對吧 :)，放心，我只寫有趣的，嚴肅的放公司Confluence裡就好。

總之，本Blog稟持10年前的精神，是我個人的社交空間，只希望各位親朋好友們玩的開心，如果有緣你進來了，還看到現在，就留個言當紀念吧 YO~
