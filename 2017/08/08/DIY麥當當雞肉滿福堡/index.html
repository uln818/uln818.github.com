食譜-DIY麥當當雞肉滿福堡

最近開始研究漢堡排的製作，由於牛肉尚未研發完成
先來發一篇雞肉的吧~

由於很常自製早餐，之前大都是使用刈包夾東西來吃，吃久了難免會覺得想膩想要換一下，結果意外的在Costco買到了義美瑪芬堡，一吃之下驚為天人啊，這簡直就跟麥當當的滿福堡一模一樣！跟大家分享一下。

食材

  1. Costco 雞腿肉一包(大概可以做4份)
  2. 義美瑪芬堡
  3. 雞蛋
  4. 起司、米酒、鹽巴、黑胡椒、迷迭香少許

做法-雞腿肉漢堡排

漢堡最精華的當然是肉排，下面先說肉排的做法：

1. 雞腿肉去皮去油後，切丁

因為我吃不慣雞胸肉，所以去Costco一律都是買雞腿肉，家裡隨時都有。漢堡排好吃與否其實跟油脂含量有關，如果想吃口感好一點，可以不要把油去得太乾淨。

2. 將切丁的雞腿肉，加入米酒、鹽巴、黑胡椒、和一點點的迷迭香，抓一抓，醃五分鐘

比例我都是一片去皮去油的雞胸肉配一大匙米酒，鹽巴轉6-8下，黑糊椒/迷迭香都只撒一點點，切記迷迭香不要太多，一點就很香了。一包會有兩片雞腿肉，比例自己調整。

3. 拿出人力/電動切碎機，把醃的雞肉變成絞肉

我妹從掏寶買的人力攪碎機，邊打還可以邊練身體喔! 哈哈

4. 拿出秤子和保鮮膜，以一塊80g-100g的重量為做成肉餅，然後包起來直接放到冷凍庫冰

剛好雞肉吃完了，圖片是牛肉，雞肉顏色不會那麼深。

雞腿肉一片大約200g，去皮去油後大約160g-180g左右，兩片外加加了米酒，最後應該會接近400g，這樣剛好以100g為單位做成四片。

作法-雞肉滿福堡

1. 把義美瑪芬堡放到電鍋一杯水蒸

2. 在蒸的同時把冷凍的雞腿肉漢堡排拿出來煎，然後順便煎一顆蛋

因為我懶，煎肉排都是直接從冷凍庫拿出來煎，放大約5g的橄欖油，兩面隨便煎，煎有一點焦 or 感覺漢堡排有湯汁流出來就好，很難失敗，沒啥技巧。記得煎到快好時，順便煎個蛋。

3. 由下而上，以 瑪芬堡>漢堡排>起司>荷包蛋>瑪芬堡 的順序包好，順序很重要! XDD

4. 完工!

如果做的順的話，大約10-15分鐘就可以做完早餐，相當方便。蒸瑪芬堡的水可以不用到一杯，我都大概八分滿而已。
